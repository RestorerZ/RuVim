В данном каталоге находятся файлы с описанием сопоставления клавиш клавиатуры к
вводимым символам.

Один из этих файлов может быть подключён к редактору Vim при указании его
наименования как значение параметра 'keymap'.

Наименование этих файлов формируется по определённым правилам, и состоит из
следующих частей:

	{language}[-{layout}][_{encoding}].vim
	{язык}[-{раскладка}][_{кодировка}].vim

Где:

{язык}		наименования языка (например, «russian», «greek»).
		Обязательная часть.

{раскладка}	наименование раскладки клавиатуры (например, «qwerty»,
		«dvorak»).  Если эта часть отсутствует, то подразумевается
		стандартная раскладка.
		Не обязательная часть.

{кодировка}	кодировка символов, с которой будет работать данное описание
		сопоставления клавиш.  Если эта часть отсутствует, то
		подразумевается «нормальная» кодировка символов для данного
		языка.
		Не обязательная часть наименования.
		Данная часть наименования файла должна указываться, так же как и
		значение параметра 'encoding' — допускаются символы только в
		нижнем регистре и символ “-” вместо символа “_” в обозначении
		самой кодировки.

Каждый файл начинается с заголовка, в котором указывается имя того, кто
обслуживает этот файл и дата последнего изменения файла.  Если обнаружится
какая-либо проблема с файлом описания сопоставления клавиш, то проверьте,
имеется ли более новая версия этого файла.  Если возможно, то сообщите о
проблемах специалисту, обслуживающему этот файл.

Формат строк, расположенных после команды "loadkeymap", описан в документации,
поставляемой с редактором Vim. Чтобы просмотреть описание, введите команду
":help keymap-file-format" (без кавычек).
